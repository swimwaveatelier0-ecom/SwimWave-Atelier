# Swimsuit Business Website

A user-friendly e-commerce website for selling swimsuits.

## Features
- Product listings with images, descriptions, and prices
- Add to cart functionality with quantity tracking
- Remove items from cart
- Persistent cart using localStorage
- Checkout simulation
- Responsive design with modern UI
- Navigation menu and footer

## How to Run
1. Open `index.html` in your web browser.
2. Browse the products and click "Add to Cart" to add items.
3. View and manage your cart in the sidebar.
4. Click "Checkout" to complete your purchase (simulated).
